 # vilt-map-se
 A browser of hunt data in Sweden

 # what am i looking at here
 This is a map of all confirmed hunting kills in sweden between 2004-2024. The idea is that it should show rises and falls in comparitive specieis levels over time. Nothing that other tools don't show in their own way but I wanted to make it. I'd also like to add report data to see how reported data maps against hunt data (does overreporting lead to overhunting?)

 # why did you make this
 I'm doing a hunting course and I've found that making a tool on the topic is fun.

 # wheres the data from
 its from the hunters federation. had to do a lot of queries (49*21) to get it as there was no easy other way.
